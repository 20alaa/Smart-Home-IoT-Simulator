Smart Home IoT Simulator

Project Overview:
A Java-based simulation of a Smart Home System that manages various IoT devices. This project demonstrates software design patterns, specifically focusing on Aggregation and Dynamic Polymorphism.

Technical:
Object Aggregation: Implements a `SmartHome` class that manages a collection of `Device` * Polymorphic Method Overriding: Features specialized behavior for smart devices (e.g., `SmartLight`) that override base class methods to provide detailed device-specific information.objects using `ArrayList`.
Encapsulation: use of private data members and public getters/setters to ensure data integrity and secure state management.

Technologies Used:
Language: Java
Core Concepts: Aggregation, Polymorphism, Encapsulation, Collections (ArrayList)
IDE: Eclipse
